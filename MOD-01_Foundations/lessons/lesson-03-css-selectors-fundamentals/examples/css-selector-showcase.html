Basic Selector Types

Element Selectors

These paragraphs are styled using element selectors.

All <p> elements get the same basic styling.

p { color: #333; line-height: 1.6; }

Class Selectors

This card uses .demo-card class styling.

This text has additional .highlight-text styling.

.demo-card { background: #f8f9fa; }

ID Selectors

The header above uses #main-header ID selector.

IDs are unique and have high specificity.

#main-header { background: linear-gradient(...); }

Attribute Selectors

Form inputs below use attribute selectors:

input[type="email"] { background: #f0fff0; }

Child Selectors

This paragraph is a direct child and gets special styling.

This nested paragraph doesn't get the child selector styling.

Hover Effects
