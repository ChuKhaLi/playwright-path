Step 1: Basic Element Selectors

Element selectors target HTML elements by their tag name. This is the most basic type of CSS selector.

h1 { color: #2c3e50; } p { margin-bottom: 15px; }
Try this: All <h1> elements on this page are styled with the same color using element selectors.

Step 2: Class Selectors for Reusable Styles

Class selectors allow you to create reusable styles that can be applied to any element.

This is a .highlight class - notice the yellow background!
This is a .warning class - red background indicates caution.
This is a .success class - green background shows success.
.highlight { background-color: #fff3cd; } .warning { background-color: #f8d7da; } .success { background-color: #d4edda; }

Step 3: Styling Forms with Attribute Selectors

Attribute selectors are perfect for styling form elements based on their type or attributes.

input[type="text"] { border: 2px solid #bdc3c7; } input[required] { border-left: 4px solid #e74c3c; } input[required]:valid { border-left: 4px solid #27ae60; }

Summary: What We've Learned

Key Selector Types Covered:

  • Element selectors: Target HTML tags directly (h1, p, div)
  • Class selectors: Reusable styles with .classname
  • ID selectors: Unique elements with #idname
  • Attribute selectors: Target based on attributes like [type="text"]
  • Pseudo-classes: Interactive states like :hover, :focus
  • Pseudo-elements: Add content with ::before, ::after
  • Combinators: Relationships like child > and descendant (space)
Congratulations! You now understand the fundamental CSS selectors needed for both styling websites and identifying elements for automated testing.
